INTERACTIVE MEDIA DESIGN

WHAT IS IDP?

Interactive Media Design (IDP) is a two-year diploma program at Fanshawe's London Downtown Campus. Learn web design, programming, 3D animation, and more. Graduate with skills for careers in design studios, software development, and media production.

IDP Program Overview

Image

WHAT DO IDP STUDENTS FOCUS ON?

With a strong foundation in web design, programming, 3D animation, and print design, they hone their skills to create captivating and interactive digital experiences. Additionally, students delve into the world of design, coding, marketing, and media management. This holistic approach ensures that IDP graduates are well-rounded professionals ready to excel in communication design, software development, interactive software publishing, media production, and IT consulting.

The program places a special emphasis on web application design, interactive rich media, and the latest technologies, blending creativity with technical expertise to craft user-friendly, media-rich websites and applications.
